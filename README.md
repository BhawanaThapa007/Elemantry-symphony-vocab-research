## 📋 Overview

This is a research-based vocabulary learning system designed for **Symphony English Grade 4** students in low-resource EFL contexts. The system implements:

- **🤖 AI-Adaptive Learning** (Zone of Proximal Development - Vygotsky)
- **🎮 Evidence-Based Gamification** (Self-Determination Theory - Deci & Ryan)
- **📊 Cognitive Load Optimization** (CTML - Mayer)
- **📖 Comprehensive Vocabulary Instruction** (Nation's Four Strands)

## 🎯 Research Framework

### **Theoretical Foundation:**

1. **Vocabulary Learning Theory** (Nation, 2007, 2013; Beck et al., 2002)
   - Tier 2 word selection
   - Four Strands implementation
   - Multiple meaningful exposures

2. **CTML - Cognitive Theory of Multimedia Learning** (Mayer, 2021)
   - Multimedia Principle (d=1.50)
   - Modality Principle (d=0.72)
   - Segmenting Principle (86% improvement)
   - Personalization Principle (d=0.79)
   - Coherence Principle (d=-0.35 violations)

3. **SDT - Self-Determination Theory** (Deci & Ryan, 2000)
   - Autonomy Support
   - Competence Development
   - Relatedness Fostering

4. **ZPD - Zone of Proximal Development** (Vygotsky, 1978)
   - Adaptive difficulty
   - Graduated scaffolding (5 levels)
   - Automatic fading

5. **DGBL - Digital Game-Based Learning** (Prensky, 2001)
   - Clear goals
   - Immediate feedback
   - Optimal challenge

6. **TAM - Technology Acceptance Model** (Davis, 1989)
   - Teacher evaluation framework
