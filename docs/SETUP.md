# 🛠️ Setup Guide

## Local Development Setup

### **1. Clone Repository**

```bash
git clone https://github.com/[username]/symphony-vocab-research.git
cd symphony-vocab-research
```

### **2. Open in Browser**

Simply open `index.html` in any modern web browser:

```bash
# Mac/Linux
open index.html

# Windows
start index.html
```

**That's it!** No build process, no dependencies.

---

## GitHub Pages Deployment

### **Enable GitHub Pages:**

1. Go to repository **Settings**
2. Click **Pages** (left sidebar)
3. Source: **Deploy from main branch**
4. Folder: **/ (root)**
5. Click **Save**
